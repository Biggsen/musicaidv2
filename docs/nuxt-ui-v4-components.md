# Nuxt UI v4 – Full Component List (with descriptions)

## Layout
- **App** — Wraps your app to provide global configurations and structure.
- **Container** — Centers and constrains the width of content.
- **Error** — A pre-built error component with NuxtError support.
- **Footer** — A responsive footer component.
- **Header** — A responsive header component.
- **Main** — A main element that fills the available viewport height.

## Element
- **Alert** — A callout to draw user’s attention.
- **Avatar** — An image element with fallback and Nuxt Image support.
- **AvatarGroup** — Stack multiple avatars in a group.
- **Badge** — A short text to represent a status or category.
- **Banner** — Display a banner at the top of your website.
- **Button** — A button element that can act as a link or action.
- **Calendar** — A calendar component for selecting dates.
- **Card** — Display content in a card with header, body, footer.
- **Chip** — Indicator of a numeric value or a state.
- **Collapsible** — Toggle visibility of content.
- **FieldGroup** — Group button-like elements together.
- **Icon** — Display any icon from Iconify or other sources.
- **Kbd** — Displays a keyboard key representation.
- **Progress** — Shows progress of a task.
- **Separator** — Separates content horizontally or vertically.
- **Skeleton** — Loading placeholder.

## Form
- **Checkbox** — Toggle between checked/unchecked.
- **CheckboxGroup** — Checklist with multiple selection.
- **ColorPicker** — Select a color.
- **FileUpload** — Upload files.
- **Form** — Form with built‑in validation.
- **FormField** — Wrapper providing validation & error display.
- **Input** — Text input.
- **InputDate** — Date selection input.
- **InputMenu** — Autocomplete input with suggestions.
- **InputNumber** — Numeric input with range.
- **InputTags** — Input with tag display.
- **InputTime** — Time selection input.
- **PinInput** — PIN code input.
- **RadioGroup** — Select a single option.
- **Select** — Select from list options.
- **SelectMenu** — Searchable select.
- **Slider** — Select numeric value via slider.
- **Switch** — Toggle between two states.
- **Textarea** — Multiline input.

## Data
- **Accordion** — Stacked collapsible panels.
- **Carousel** — Swipeable carousel using Embla.
- **Empty** — Display an empty state.
- **Marquee** — Infinite scrolling content.
- **Table** — Responsive data table.
- **Timeline** — Event timeline with dates.
- **Tree** — Tree view for hierarchical data.
- **User** — User profile display.

## Navigation
- **Breadcrumb** — Hierarchy of links.
- **CommandPalette** — Full‑text search palette (Fuse.js).
- **FooterColumns** — List of footer link columns.
- **Link** — Wrapper around NuxtLink with extras.
- **NavigationMenu** — Horizontal/vertical navigation links.
- **Pagination** — Page navigation buttons/links.
- **Stepper** — Multi‑step progress indicator.
- **Tabs** — Tab panels displayed one at a time.

## Overlay
- **ContextMenu** — Right‑click menu.
- **Drawer** — Slide‑in drawer.
- **DropdownMenu** — Click‑triggered menu.
- **Modal** — Dialog window.
- **Popover** — Floating, non‑modal dialog.
- **Slideover** — Screen‑side sliding dialog.
- **Toast** — Feedback message.
- **Tooltip** — Hover information popup.

## Page
- **AuthForm** — Login/register/reset form.
- **BlogPost** — Article layout.
- **BlogPosts** — Blog post grid layout.
- **ChangelogVersion** — Changelog article.
- **ChangelogVersions** — Timeline of changelog versions.
- **Page** — Grid layout with side columns.
- **PageAnchors** — Anchor navigation list.
- **PageAside** — Sticky page navigation sidebar.
- **PageBody** — Main page content.
- **PageCard** — Pre‑styled card for page content.
- **PageColumns** — Multi‑column content layout.
- **PageCTA** — Call‑to‑action section.
- **PageFeature** — Highlight application features.
- **PageGrid** — Responsive content grid.
- **PageHeader** — Responsive page header.
- **PageHero** — Hero section.
- **PageLinks** — List of links.
- **PageList** — Vertical stacked list layout.
- **PageLogos** — Logo/image list.
- **PageSection** — Page content section.
- **PricingPlan** — Individual pricing plan.
- **PricingPlans** — Grid of pricing plans.
- **PricingTable** — Tiered pricing table.

## Dashboard
- **DashboardGroup** — Layout wrapper for dashboards.
- **DashboardNavbar** — Responsive navbar for dashboard.
- **DashboardPanel** — Resizable dashboard panel.
- **DashboardResizeHandle** — Sidebar/panel resizing handle.
- **DashboardSearch** — Command palette for searching dashboard content.
- **DashboardSearchButton** — Button to trigger dashboard search.
- **DashboardSidebar** — Collapsible sidebar.
- **DashboardSidebarCollapse** — Collapse sidebar (desktop).
- **DashboardSidebarToggle** — Toggle sidebar (mobile).
- **DashboardToolbar** — Toolbar under navbar.

## Chat
- **ChatMessage** — Chat message with avatar & actions.
- **ChatMessages** — List of chat messages.
- **ChatPalette** — Chat UI overlay.
- **ChatPrompt** — Enhanced textarea for chat prompts.
- **ChatPromptSubmit** — Submit button with status indicators.

## Content
- **ContentNavigation** — Accordion navigation for docs.
- **ContentSearch** — Search palette for documentation.
- **ContentSearchButton** — Button to open content search.
- **ContentSurround** — Prev/next navigation for pages.
- **ContentToc** — Sticky table of contents with active link tracking.

## Color Mode
- **ColorModeAvatar** — Avatar with light/dark sources.
- **ColorModeButton** — Light/dark toggle button.
- **ColorModeImage** — Image with light/dark sources.
- **ColorModeSelect** — Select for system/light/dark.
- **ColorModeSwitch** — Toggle switch for color mode.
